------- 워크래프트3 버전 간 좌표 변환 --------           
[Note]에서 좌표 변환 시 UI 영역을 지정합니다.변환이 필요 없다면 생략 가능합니다.
#panel=items# 인벤토리 + 상점
#panel=skill# 스킬창
#panel=map#  미니맵
#panel=cmd#  명령창 (이동, 정지 등)

클래식 워3는 UI 가 창 크기에 정 비례하기 때문에 아무 해상도나 변환 가능합니다.
리포지드는 16:9 기준으로 작성 되었고, 창 비율이 바뀌면 UI 영역을 수정 해야합니다.

c_map 폴더:
고정 좌표 및 명령어에 이름을 지정해 재사용할 수 있도록 저장하는 폴더입니다.
Warcraft III의 각 버전(w3_ver)에 맞춰 동일한 이름으로 작성해두면,
Read: 명령에서 경로만 바꿔 쉽게 변환할 수 있습니다
